# Todo Task Card

A responsive, accessible todo/task card implemented with HTML, CSS, and JavaScript.

## Features

- Semantic markup with `article`, `h2`, `p`, `time`, `label`, `button`, and `role="list"`
- Required `data-testid` attributes for automated tests
- Accessible checkbox with visible label
- Live-friendly time remaining display
- Responsive layout for mobile and desktop
- Keyboard-focusable `Edit` and `Delete` buttons

## Run

Open `index.html` in a browser.
